# Project Management System

A simple and efficient web application that helps users manage projects and tasks in one place.

The system allows users to create projects, add tasks, track progress, and monitor overall productivity through an easy-to-use dashboard.

---

## ✨ Features

### 👤 User Authentication
- User Registration
- Secure Login
- Logout Functionality
- Password Encryption for Security

### 📁 Project Management
- Create New Projects
- View Project Details
- Edit Existing Projects
- Delete Projects
- Track Project Status
  - Not Started
  - In Progress
  - Completed

### ✅ Task Management
- Create Tasks Inside Projects
- Update Task Information
- Delete Tasks
- Mark Tasks as Completed
- Set Task Priorities
  - Low
  - Medium
  - High
- Track Task Status

### 📊 Dashboard
The dashboard provides a quick overview of:

- Total Projects
- Total Tasks
- Completed Tasks
- Pending Tasks
- Projects Currently In Progress

### 🔒 Security Features
- Secure User Authentication
- Password Hashing
- Protected Routes
- JWT Authentication
- Input Validation
- Rate Limiting for Login Requests

---

## 🎯 Purpose of the Project

Managing multiple projects and tasks manually can be difficult.

This Project Management System was developed to provide a centralized platform where users can:

- Organize their work efficiently
- Track project progress
- Manage tasks effectively
- Improve productivity
- Monitor overall performance through statistics
